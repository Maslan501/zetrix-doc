# Getting Help

If you encounter issues not covered here:

1. **Check Documentation**
   * [GitHub Repository](https://github.com/Zetrix-Chain/zetrix-mcp-server)
   * [Official Zetrix Docs](https://docs.zetrix.com/)
2. **Search GitHub Issues**
   * https://github.com/Zetrix-Chain/zetrix-mcp-server/issues
3. **Create New Issue**
   * Include error messages
   * Describe steps to reproduce
   * Provide configuration (redact sensitive info)
4. **Community Support**
   * Zetrix Discord/Telegram
   * MCP Community Forums
