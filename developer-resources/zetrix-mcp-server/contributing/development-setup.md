# Development Setup

#### **Fork and Clone**

```bash
git clone https://github.com/YOUR_USERNAME/zetrix-mcp-server.git
cd zetrix-mcp-server
```

#### **Install Dependencies**

```bash
npm install
```

#### **Build Project**

```bash
npm run build
```

#### **Run Tests**

```bash
npm test
```

#### **Run in Development Mode**

```bash
npm run dev
```
