# 10.2 Glossary

| Term | Meaning |
| --- | --- |
| **x401** | Identity proof protocol (401 Proof-Required). |
| **x402** | Payment protocol (402 Payment Required). |
| **VC / VP** | Verifiable Credential / Verifiable Presentation. |
| **DID / ZID** | Decentralised Identifier (Zetrix variant). |
| **DCQL** | The verifier's claim query language. |
| **BBS+** | Signature scheme enabling selective disclosure. |
| **Bulletproof** | Zero-knowledge range proof. |
| **Holder binding** | Ed25519 signature over the verifier nonce. |
| **HSM** | Hardware-grade key store. |
| **Facilitator** | x402 settlement service. |
| **JMYR** | A ZTP20 asset used for settlement. |
